# Potato Rescue - LOVE Port

This repository still contains the original Godot prototype, but the playable LOVE version now lives at the project root.

## Run

```sh
love .
```

## Controls

- Move: `A/D` or arrow keys
- Jump / double jump: `Space`, `W`, or up arrow
- Restart current level: `R`
- Quit: `Esc`

## Game

Poti is a potato hero trying to rescue Princess Spudella from roaming potato peelers. Each level has a key and a door. Collect chips for bonus score, stomp peelers from above, avoid spikes, and reach the final princess rescue.

The game includes a simple procedural relaxing music loop generated at startup, so no extra audio files are required.
